# AGENTS.md

This repository is a public starter for HubSpot agent tools.

## Mission

Help people build useful HubSpot agent tools without assuming they are full-time developers.

## Rules

- Keep the repo generic and public-safe.
- Do not commit secrets, real account IDs, or customer data.
- Do not hardcode one hosting provider.
- Keep examples understandable before making them clever.
- Prefer explicit placeholders over hidden assumptions.
- If you add auth complexity, document whether it belongs here or in a separate OAuth service.

## Repo intent

This repo should help users scaffold custom HubSpot tools quickly.
It should stay simple enough for AI coding tools to read and modify safely.

## AI editing guidance

- Keep README beginner-friendly.
- Keep comments short and practical.
- Mark placeholder logic clearly.
- Document assumptions when adding integrations.
- If one hosting provider is mentioned, also note that alternatives are valid.
